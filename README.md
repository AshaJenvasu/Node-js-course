# 🚀 My Journey to the Backend with Node.js

This repository is a dedicated space for my self-study journey into Backend Development. As a **Lifelong Learner** coming from a Frontend background (React/TS), I'm expanding my horizons to become a Full-Stack Engineer.

## 📚 What I'm Learning

Currently following the intensive course by **Dave Gray** , focusing on NodeJS, the Express JS framework, and MongoDB.

## 🛠️ Tech Stack & Skills

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB / PostgreSQL (via Prisma ORM)
- **Security:** JWT Authentication (Access/Refresh Tokens), Password Hashing (Bcrypt)
- **DevOps:** Dockerization (Containerizing Node & DB)

## 📖 Key Milestones

- [ ] Initializing a Web Server & Custom Logging
- [ ] Mastering MVC Pattern & Route Handlers
- [ ] Implementing Secure JWT Authentication Flow
- [ ] Integrating Prisma ORM with Relational Databases
- [ ] Dockerizing the entire ecosystem
